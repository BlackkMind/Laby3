
Tak pokrótce program tworzy tablice 200 losowych elementów. Na samym początku program prości użytkowinka o wybranie metody sortowania.
Sortowanie Bombelkowe jest zawartę w klasie  bombelki gdzie kopiowania jest oryginalna tablica aby nie zmieniać oryginały. Następnie program porównuje elementy tablicy i ustala je wdług zadanego porządku.
Sortowanie Kubełkowe jest zawartę w klasie wiaderka gdzie z założenie są utworzone 5 wiaderek (choć im więcej ich zadeklarujemy tym mniej prównań będzie) następnie jest szukana maksymalna wartość w tablicy i tworzona zostaje lista dla każdego kubełka i rozdzialne są kolejne elementy na kubełki
Zawartość poszczególnych kubełków zostaje posortowania za pomocą sortowania bombelkowego i wyświetlona.
